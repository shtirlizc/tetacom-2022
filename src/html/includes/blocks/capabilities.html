Возможности

Проектирование дизайнов

При проектировании дизайна формируется поэтапный сценарий проведения ГТМ по кислотной обработке с учетом различных технологических режимов и способов закачки, осуществляется выбор наилучших рецептур составов из библиотеки реагентов Симулятора, содержащей информацию по проведенным физико-химическим и фильтрационным лабораторным исследованиям.

Построение геолого-физической модели пласта

Формирование входных данных по скважине осуществляется с помощью автоматизированной загрузки данных по конструкции скважины, а также данных геофизических и гидродинамических исследований. В Симуляторе реализована возможность построения 1D-геомеханической и петрофизической моделей пласта.

Анализ тестовых закачек

Для калибровки и адаптации дизайна и математической модели в Симуляторе представлен комплекс инструментов для анализа тестовых закачек для кислотных обработок (оценка трений, Хорнера, SRT и пр.). и матчинга проведенных работ по фактическим данным закачки.
