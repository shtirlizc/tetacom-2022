## Задачи

- Разработка продукта в нефтегазовой сфере (работа со сложными таблицами и графиками)
- Развитие существующей библиотеки компонентов
- Развитие корпоративного сайта
- Взаимодействовать с дизайнерами, бэкенд разработчиками, тестировщиками
- Работать по SCRUM (2-х недельные спринты)

## Мы ожидаем

- Опыт в области разработки от 2 лет
- Хорошего знания JavaScript и TypeScript
- Хорошего знания Angular (от 10 версии), RxJS
- Хорошего знания HTML5 и CSS3
- Опыта работы с CSS препроцессорами (SCSS, LESS)
- Уверенной работы с адаптивной и кроссбраузерной версткой
- Опыта разработки UI компонент из Figma
- Опыта работы со сложным REST API
- Опыт с Git
- Опыт разработки SPA
- Знание английского языка (чтение технических документов и профессиональной литературы)
- Плюсом будет опыт работы с Docker (на базовом уровне), опыт работы с d3, three.js, cubejs

## Мы предлагаем

- Трудоустройство по ТК РФ
- Обучение за счёт компании
- Свободу действий в принятии решений
- Профессиональную команду
- Возможность развиваться вместе с нашей компанией
- Достойную заработную плату
