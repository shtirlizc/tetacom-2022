Рассмотрен пример сервиса моделирования в Симуляторе ОПЗ "RockStim" дизайна двухстадийной солянокислотной обработки с применением отклонителя с используемыми фактическими данными по расходу и давлению с устьевых датчиков с последующей оценкой сходимости результатов.

### Исходные данные

**Тип коллектора:** карбонатный

**Пластовая температура:** 37°С

**Конструкция скважины:** вертикальная

**Тип скважины:** нефтедобывающая

**Режим обработки:** закачка с пакером

**Модель расчета:** 2D

**Используемые реагенты:** водосолевая эмульсия, кислота (15% HCl)

### Моделирование

Скин-фактор определен с помощью блока **"Анализ добычи"** с использованием фактических режимных данных работы скважины.

Перед кислотной обработкой проведены лабораторные исследования по определению кинетических параметров реакции кислоты с горной породой, которые были использованы при моделировании.

Согласно фактическим данным сформирован план закачки:

**Отклонитель 15м³ → Кислота 9,99м³ → Отклонитель 5м³ → Кислота 9,99м³ → Продавка нефтью 5,81м³**

Для выполнения качественного моделирования ОПЗ была выбрана 2D-модель (сетка 20х20м), червоточины моделировались согласно модели Гонга с максимальной детализацией выходных данных.

### Симуляция закачки

По результатам моделирования отмечается проникновение отклонителя в высокопроницаемую часть продуктивного интервала, что позволят выровнять в некоторой степени профиль проникновения кислотного состава.

### Результаты

**Скин-фактор:** до **+0,79** / после **-2,65**

**К-т продуктивности по жидкости (м³/(атм·сут):** до **0,38** / после **0,67**

**Дебит нефти по дизайну (м³/сут):** до **22,64** / после **38** / факт **32**

По результатам сервиса моделирования *отмечается высокая точность выполненных расчетов п*о сходимости с фактическим дебитом жидкости после ОПЗ.
